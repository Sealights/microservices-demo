Sealights Boutique Project

Welcome to the Sealights Boutique Project!

This project is designed to be your one-stop shop for learning about Sealights and its powerful features.

What You'll Get:

  • Boutique GitHub Repository
  • Pre-Configured EC2 Instance
  • Boutique Running (Port 8081)
  • Jenkins with Prepared Jobs (Port 8080)
  • Sealights Account (if needed)
  • Comprehensive Documentation

Getting Started Tips:

Read the Documentation: We recommend starting with the documentation for a comprehensive understanding of the project.

Need Help? Contact Customer Success: Our friendly customer success team is here to answer your questions.

Experimentation Encouraged: This project is your playground! Feel free to experiment without worrying about breaking your CI.
